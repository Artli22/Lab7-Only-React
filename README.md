# Demo Metal Gear Solid 6 + Snake
## Esta es una colaboracion de Metal Gear con el clasico juego de Snake

### Instrucciones
#### Controlas al soldado Venom Snake, con el objetivo de recolectar todas las manzanas repartidas alrededor del grid, cada manzana que recolectes sumara un +1 en el puntuaje y agregara una caja sigilosa al final de la fila.
#### El personaje se movera automaticamente, en linea recta acorde a la direccion del ultimo input. Controlas al personaje con las teclas w, s, a, d para moverlo arriba, abajo, izquierda y derecha respectivamente.
#### Pierdes la partida en el momento que el personaje sobrepase los bordes del grid; mostrando una pantalla de Game Over y un boton para reiniciar la partida.

### Tecnologias Implementadas
- React
- React DOM
- JS babel
